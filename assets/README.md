# Bhavya Singh | Premium Portfolio

A futuristic, high-performance portfolio website built with a custom glassmorphism design system. This repository showcases my journey as a Computer Science Engineer, Data Analyst, and AI Enthusiast.

## ✨ Key Features
- **Glassmorphism UI**: A premium, frosted-glass aesthetic with deep purple and midnight navy accents.
- **Dynamic Stats Counter**: Interactive counters for CGPA, Projects, and Certifications that trigger on scroll.
- **Project Grid**: Organized showcase of core projects with hover effects and detailed information.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Interactive Elements**: Custom cursor, scroll-reveal animations, and smooth navigation.

## 🛠️ Tech Stack
- **Core**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Icons**: [Lucide Icons](https://lucide.dev/)
- **Typography**: Google Fonts (Outfit & Inter)
- **Animation**: Intersection Observer API for scroll-reveal effects.
